# Raspberry Pi IoT Hub Resources

## Knowledge

- [FastAPI Documentation](https://fastapi.tiangolo.com/)
  Official FastAPI documentation. Use for: setting up FastAPI, writing APIs, request validation, and deployment.
- [SQLite Documentation](https://www.sqlite.org/index.html)
  Official SQLite documentation. Use for: understanding SQLite constraints, indexing, and syntax.
- [Eclipse Mosquitto MQTT Broker](https://mosquitto.org/)
  Official Mosquitto documentation. Use for: installing and configuring MQTT Broker on Raspberry Pi.
- [paho-mqtt Python Client](https://eclipse.dev/paho/index.html)
  Python client library for MQTT. Use for: integrating MQTT subscriber/publisher with FastAPI/Python.

## Wisdom (Communities)

- [r/raspberry_pi](https://www.reddit.com/r/raspberry_pi/)
  Reddit community for Raspberry Pi projects, troubleshooting, and optimization tips.
- [r/FastAPI](https://www.reddit.com/r/FastAPI/)
