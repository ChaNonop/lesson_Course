# กำหนดเป้าหมายและแผนการเรียนรู้ของระบบ IoT

ผู้เรียนได้กำหนดภารกิจการสร้างระบบ Local IoT บน Raspberry Pi Zero 2 W โดยเน้นความปลอดภัยและมีหน้าเว็บสีเขียวพาสเทลโมเดิร์น นอกจากนี้ ได้ระบุความต้องการในการเน้นการใช้ AI เขียน Prompt เจน JavaScript เพื่อใส่ใน Function Node ของ Node-RED แทนการสอนการลากเส้นจูนบล็อกธรรมดา ซึ่งจะช่วยกำหนดทิศทางการสอนให้ตรงเป้าหมายที่สุด
