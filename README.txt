========================================================================
    IoT Hub Curriculum: พัฒนาระบบเครือข่ายฐานข้อมูลและฮาร์ดแวร์ IoT
========================================================================

โปรเจกต์นี้เป็นหลักสูตรการสอนและพัฒนาเซิร์ฟเวอร์ IoT Hub (หลักสูตร 10 บทเรียน) 
เพื่อสร้างระบบรวบรวมข้อมูลเซ็นเซอร์จากบอร์ดไมโครคอนโทรลเลอร์ (ESP32/ESP8266) 
เก็บเข้าสู่ฐานข้อมูล SQLite บนเครื่องเซิร์ฟเวอร์ และแสดงผลผ่านเว็บบอร์ดแสดงผลที่ตอบสนอง (Responsive Dashboard)
รวมถึงเทคนิคการจัดการระบบเครือข่ายและความมั่นคงปลอดภัยขั้นสูง

------------------------------------------------------------------------
 โครงสร้างบทเรียนหลักสูตร (Lessons Overview)
------------------------------------------------------------------------

[หมวดหมู่บทเรียนหลัก - Core Lessons]
- บทเรียนที่ 1: ภาพรวมสถาปัตยกรรม IoT Hub (ESP32 -> MQTT -> FastAPI -> SQLite -> Web UI)
- บทเรียนที่ 2: การควบคุม Terminal, tmux และการตั้งค่าความปลอดภัย MQTT Broker
- บทเรียนที่ 3: พื้นฐาน Docker และ Docker Compose ในงาน IoT Hub
- บทเรียนที่ 4: การจัดทำระบบ FastAPI, พัฒนาตารางข้อมูลด้วย ORM (SQLAlchemy) และ SQLite WAL Mode
- บทเรียนที่ 5: การผสานการทำงานของ MQTT Client (paho-mqtt) เข้ากับระบบหลังบ้าน FastAPI
- บทเรียนที่ 6: หน้าเว็บบอร์ดแสดงผลแบบตอบสนอง (Responsive Web Dashboard & Chart.js)
- บทเรียนที่ 7: การเข้าถึงระยะไกล (Inbound Access) ด้วย Cloudflare Tunnel และระบบ VPN (Tailscale)
- บทเรียนที่ 8: พื้นฐานการเขียนโปรแกรมเชิงวัตถุ (OOP) และหลักการ 4 เสาหลัก (C++ / Python)
- บทเรียนที่ 9: พัฒนาเฟิร์มแวร์ ESP32/ESP8266 บน PlatformIO แบบ Non-blocking และ OOP Sensor

[หมวดหมู่เนื้อหาเพิ่มเติม - Supplementary Content]
- บทเรียนที่ 10: การสร้างระบบเครือข่ายและความปลอดภัยส่วนบุคคล (PiVPN, Tailscale และ Pi-hole DNS Sinkhole)

------------------------------------------------------------------------
 เทคโนโลยีที่ใช้ภายในหลักสูตร (Technology Stack)
------------------------------------------------------------------------
1. Hardware / Firmware: บอร์ด ESP32, บอร์ด ESP8266, PlatformIO IDE (VS Code extension)
2. Network / Protocols: MQTT Protocol (Eclipse Mosquitto Broker), HTTP API REST, P2P WireGuard (Tailscale), Cloudflare Tunnel
3. Backend / API: Python, FastAPI Framework, SQLAlchemy ORM
4. Database: SQLite Database (เปิดใช้ WAL Mode เพื่อความสามารถการเขียน/อ่านแบบ Concurrent)
5. Front-End: HTML5, Vanilla CSS (Dual-Accent Technical Dark Theme), Vanilla JavaScript, Chart.js สำหรับวาดกราฟ

------------------------------------------------------------------------
 วิธีการเปิดเรียนรู้บนระบบ Local / Deploy
------------------------------------------------------------------------
เนื่องจากหลักสูตรนี้แสดงผลผ่านเว็บเพจสถิต (Static Web Page) คุณสามารถเรียนรู้ได้โดย:
1. เข้าไปที่โฟลเดอร์ `lessons/`
2. ดับเบิลคลิกไฟล์ `0001-architecture-overview.html` เพื่อเข้าเรียนบทที่ 1 
3. แถบเมนูด้านซ้ายและปุ่มนำทางย้อนกลับ/ถัดไปด้านล่าง จะถูกรันด้วย `main.js` เพื่อเปลี่ยนบทเรียนและหมวดหมู่ให้อัตโนมัติ

[การ Deploy ขึ้น Github Pages หรือ Host อื่นๆ]
- สามารถนำไฟล์ทั้งหมดในโปรเจกต์ (ยกเว้นเอกสารคำศัพท์เบื้องหลัง .md และโฟลเดอร์บ็อต .agents) อัปขึ้น GitHub และเปิดใช้ฟีเจอร์ GitHub Pages ชี้แหล่งต้นทางมาที่โฟลเดอร์หลัก (Root folder) เพื่อแสดงผลแดชบอร์ดบทเรียนออนไลน์ได้ทันที
